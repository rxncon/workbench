Welcome to the rxncon workbench

The workbench is a tool for working with rxncon models. If you are new to rxncon ("reaction-con"), you can learn more about it on rxncon.org.

The workbench integrates the rxncon compiler with a local model repository under a single GUI. For a brief introduction to the workbench, please read the getting started page.

The rxncon compiler reads rxncon models and compiles them into graphical or executable formats. Currently, the compiler has three graphical (Reaction graph, elemental species-reaction graph, regulatory graph) and two executable (bipartite Boolean model, rule-based model) outputs.
