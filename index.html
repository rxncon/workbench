rxncon - the reaction-contingency language

The rxncon language is a scalable and composable approach to build mechanistic models of signal transduction.

...

Network reconstruction in biology is an iterative process to formalise the structure of a biological network, reflecting our current knowledge. While reconstruction methods for metabolic networks are well established, only few reconstruction methods exist for cellular signal transduction networks. Furthermore, reconstruction of signalling networks is challenging, because state combinations lead to a large number of specific states, generating highly complex structures.

Building a rxncon model is writing down molecular biology. This process is straightforward when the biology is well known.

...

The rxncon-language enables a systematised and condensed reconstruction of cellular system by separating the reconstruction into reactions (rxn) and contingencies (con). It largely avoids the combinatorial problem by describing the network on the same granularity as the biological findings, but creates a gap between the reconstruction and established modelling methods. We developed a framework interpreting the rxncon-language and translating the reconstruction into mathematical models or visualisations. The user defines the network with one reaction list and one contingency list. From these data mathematical and graphical representation can be generated. The network can be modified and extended iteratively, and both visualisation and mathematical models can be generated automatically at any time. List of functions

The rxncon workbench integrates the rxncon toolbox with a local repository into a model managing system under a single GUI. You can download both here.

The rxncon tools are developed open source and released under the LGPL license.
